---
title: Okta Device Posture Checks
---

Okta device posture checks are security-related device attributes evaluated as part of authentication to Okta. For example, check whether a specific operating system version is installed on a device before that device can be used to access Okta. With this feature, we can define specific device posture conditions that reflect the unique security requirements of GitLab. This allows us to only access to sensitive resources only when a device meets a precise set of conditions.
