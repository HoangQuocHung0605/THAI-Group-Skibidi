## Instances

### Zendesk Global

This is our main (and first) Zendesk instance. Most people interacting with GitLab Support will use this instance.

- Accessible by: Everyone
- Production domain: `gitlab`
  - Portal access: https://support.gitlab.com/hc
  - Agent access: https://gitlab.zendesk.com/agent
  - Internal brand access: https://gitlab-internal.zendesk.com/hc
    - NOTE: Must sign-in to the agent access link first
- Sandbox domain: `gitlab1707170878`
  - Portal access: https://gitlab1707170878.zendesk.com/hc
  - Agent access: https://gitlab1707170878.zendesk.com/agent
  - Internal brand access: https://gitlab1707170878-internal.zendesk.com/hc
    - NOTE: Must sign-in to the agent access link first

### Zendesk US Government

This is a Zendesk instance made specifically with Self-Managed public sector users in mind. It has many restrictions and is not generally usable without pre-approval.

- Accessible by: US Citizens only
- Production domain: `gitlab-federal-support`

  - Agent access: https://gitlab-federal-support.zendesk.com/agent
